## What Ralph does

<CardGrid>
  <Card title="Runs Claude for days" icon="rocket">
    One binary, one per-repo daemon, many subprocesses. Survives
    compactions, OOMs, and operator terminal closes.
  </Card>
  <Card title="Ten personalities" icon="star">
    Green smashes, red triages, fixit plans, professor reflects,
    savage burns budget. Pick your variant or let fixit recommend.
  </Card>
  <Card title="Plan-DAG backed" icon="document">
    Plans live in SQLite — a deliberate state machine, not a
    markdown TODO list. Auto-backed up. Git-diffable dumps.
  </Card>
  <Card title="MCP-native" icon="setting">
    Portable stdio mode or durable HTTP daemon. Outer Claude
    orchestrates ralphs via MCP tools.
  </Card>
</CardGrid>

## Install

One package, three packagers:

```bash
# macOS / Linux / WSL2+Linuxbrew
brew tap jbcom/tap
brew install radioactive-ralph

# Windows — Scoop
scoop bucket add jbcom https://github.com/jbcom/pkgs
scoop install radioactive-ralph

# Windows — Chocolatey
choco install radioactive-ralph
```

See the full [jbcom/pkgs package index](https://jonbogaty.com/pkgs/) for
version and architecture detail.
