# Ralph has many forms

radioactive-ralph ships as a family of Claude Code skills. Each Ralph is a real operating mode: distinct model tiering, distinct parallelism, distinct tool posture, distinct stopping condition, distinct flavor of chaos.

## Pick your little guy

| Variant | What matters fast | Lore in one breath | Gate |
|---|---|---|---|
| [`/green-ralph`](./green-ralph/README.md) | Default unlimited loop across configured repos | The classic green bruiser, all feeling and no ceiling | — |
| [`/grey-ralph`](./grey-ralph/README.md) | Haiku-only repo hygiene | The original grey hustler who just wants the paste and the paycheck | — |
| [`/red-ralph`](./red-ralph/README.md) | Single-pass incident response | Principal Skinner became a Ralph and brought a clipboard | — |
| [`/blue-ralph`](./blue-ralph/README.md) | Read-only review | Cosmic museum Ralph who looks with his eyes and not his hands | — |
| [`/professor-ralph`](./professor-ralph/README.md) | Opus plan, sonnet execution, sonnet reflection | Therapy worked, mostly | — |
| [`/joe-fixit-ralph`](./joe-fixit-ralph/README.md) | Fixed-cycle, ROI-scored bursts | Tiny noir enforcer in it for the Squishee | — |
| [`/immortal-ralph`](./immortal-ralph/README.md) | Persistent recovery-first loop | The one who comes back through Miss Hoover's closet | — |
| [`/savage-ralph`](./savage-ralph/README.md) | 10 parallel agents, tier escalation, zero sleep | Pre-verbal trauma with no off switch | `--confirm-burn-budget` |
| [`/old-man-ralph`](./old-man-ralph/README.md) | Forceful one-pass imposition | A hundred-year tyrant with Wiggle Puppy in one hand | `--confirm-no-mercy` |
| [`/world-breaker-ralph`](./world-breaker-ralph/README.md) | All-opus everything | Grief weaponized into playground geology | `--confirm-burn-everything` |

## Operating profile matrix

| Variant | Parallelism | Default model posture | Scope | Typical job |
|---|---|---|---|---|
| green-ralph | 6 | mixed | configured repos | run the normal autonomous loop |
| grey-ralph | 1 | haiku only | current repo | clean docs, frontmatter, governance |
| red-ralph | 8 | sonnet with opus escalation | configured repos or one repo | fix CI failures and PR blockers |
| blue-ralph | 4 | sonnet | configured repos | review without modifying |
| professor-ralph | 4 | opus → sonnet → sonnet | configured repos | think first, then execute |
| joe-fixit-ralph | 1 | haiku/sonnet | one repo | do a few high-ROI tasks |
| immortal-ralph | 3 | sonnet only | configured repos | keep working through failures |
| savage-ralph | 10 | +1 tier escalation | configured + discovered repos | clear a backlog fast |
| old-man-ralph | 3 | sonnet/opus | targeted branch state | impose a chosen outcome |
| world-breaker-ralph | 10 | opus only | all repos | treat every task like an emergency |

## How to install

```bash
# one skill
ralph install-skill --variant green-ralph

# or all of them
ralph install-skill --all
```
