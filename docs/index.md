## Why this docs site feels different

- **One source of truth:** the root README is for GitHub/PyPI, the skill READMEs live in `skills/`, and the docs curate the story without shadow copies.
- **Brand-first Shibuya:** Ralph green is the accent, Ralph fonts are local, and the layout is custom instead of stock.
- **Variant-aware navigation:** every Ralph has a proper landing page with lore, operations, and arguments above the fold.
